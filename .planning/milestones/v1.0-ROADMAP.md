## Overview

This milestone transformed `easyk6` from a small JavaScript demo into a recruiter-ready k6 browser framework that reuses `easyPlaywright` as its permanent Playwright POM source. Intentionally smoke-first: prove the adaptation, make it runnable and understandable, then keep load/capacity as example depth without letting them dominate the repo.

The reference repo `ir-perf-k6` informed the structure, but not every feature survived into the showcase. Each phase stripped the idea down to the clearest story a reviewer can understand: upstream sync, k6-safe adaptation, smoke execution, example scaling paths, polished documentation.

**Key Decisions:**

- [Init] `easyPlaywright` is the permanent upstream Playwright POM model; `ir-perf-k6` is a pattern source, not a blueprint to copy wholesale
- [Init] Smoke is the supported first-class workflow; load and capacity remain example code paths in this milestone; Grafana work explicitly deferred
